Windows 10のサポート終了（2025年10月）が近づくにつれ、「そろそろWindows 11に移行したい」という方が増えています。しかし、すべてのPCがWindows 11に対応しているわけではありません。移行前に必ず確認すべきポイントを、わかりやすく解説します。

## Windows 11の動作要件

Microsoftが定めるWindows 11の最低要件は以下の通りです。

| 項目 | 最低要件 |
|------|----------|
| プロセッサ | 1GHz以上、2コア以上（対応リスト内のもの） |
| RAM（メモリ） | 4GB以上 |
| ストレージ | 64GB以上の空き容量 |
| TPM | TPM 2.0（必須） |
| ファームウェア | UEFI、セキュアブート対応 |
| ディスプレイ | 720p以上、9インチ以上 |

特に注意が必要なのが **TPM 2.0** と **対応プロセッサ** です。2017年以前に製造されたPCの多くは、これらの要件を満たさず、公式にはアップグレード対象外となります。

---

## PC正常性チェックアプリで確認する

自分のPCがWindows 11に対応しているかどうかは、Microsoftが提供する「PC 正常性チェック」アプリで簡単に確認できます。

**確認手順：**

1. Microsoft公式サイトから「PC 正常性チェック」アプリをダウンロード・インストール
2. アプリを起動し、「今すぐチェック」をクリック
3. 結果に「このPCはWindows 11の要件を満たしています」と表示されれば移行可能

「要件を満たしていません」と表示された場合、理由が項目別に表示されます。「TPM 2.0が必要です」「プロセッサがサポートされていません」など、具体的な原因がわかります。

---

## TPM 2.0が無効な場合

PCのハードウェアとしてはTPM 2.0を搭載しているが、BIOS/UEFI設定で無効になっているケースがあります。この場合は設定を有効にするだけでクリアできます。

**確認・有効化の流れ：**

1. PC起動時に `Del` キーや `F2` キーでBIOS/UEFI画面を開く
2. セキュリティ設定の中にある「TPM」または「PTT（Intel）」「fTPM（AMD）」を探す
3. 「Enabled（有効）」に変更して保存・再起動

ただし、BIOS操作は誤ると起動できなくなるリスクもあるため、不慣れな方は無理に行わず専門店に相談することをおすすめします。

## 移行前にやっておくべき準備

### 1. バックアップをとる

アップグレード中にトラブルが発生した場合、データが消える可能性があります。外付けHDDやクラウドに重要なデータをバックアップしてから移行してください。

### 2. ドライバーの互換性を確認する

プリンターや古い周辺機器のドライバーが、Windows 11に対応していない場合があります。メーカーのサポートページで対応状況を確認しておきましょう。

### 3. ソフトウェアの対応を確認する

業務で使っている専用ソフトや古いアプリが、Windows 11で動作するか事前に確認しておきましょう。特に、古い会計ソフトや産業機器の管理ソフトには非対応のものがあります。

---

## 移行後の注意点

- **スタートメニューやタスクバーの配置が変わる** → 慣れるまで少し時間がかかります
- **一部の設定の場所が変わる** → コントロールパネルの機能が「設定」アプリに統合されています
- **互換性の問題が出たら** → 「プログラム互換性トラブルシューティング」を使うか、管理者として実行してみましょう

---
