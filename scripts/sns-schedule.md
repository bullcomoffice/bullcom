2. X 投稿画面に貼り付け
3. サムネ画像を添付:
   - エクスプローラーで `D:/Data/Projects/Next/bullcom/public/blog-thumbnails/` を開く
   - **サムネ**: `{ファイル名}.jpg` をドラッグ＆ドロップ
4. 投稿

### 注意
- **時刻**: 09:00 JST 厳守（ブログ公開と同タイミング）
- **OGPカード**: 投稿後すぐは画像カードが出ない場合あり（X側のクロール待ち）
- **URL**: t.co短縮されるので140字計算には23字相当でカウント
- **画像つき投稿は完全無料**（API使わない手動投稿のため）
